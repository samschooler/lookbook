# Lookbook Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a lightweight macOS RAW photo editor with LUT finishing — load camera RAW files, edit with basic controls, apply .cube LUTs with thumbnail previews, and export JPEG.

**Architecture:** SwiftUI app with a three-region HSplitView (image preview, edit sliders, LUT browser). The image pipeline is a lazy CIImage filter chain: CIRAWFilter → CIColorControls → CIColorCubeWithColorSpace → CIDissolveTransition. Preview rendering uses MTKView via NSViewRepresentable with CIRenderDestination for GPU-backed display. All edits are non-destructive.

**Tech Stack:** Swift, SwiftUI, Core Image (CIRAWFilter, CIColorControls, CIColorCubeWithColorSpace), Metal (MTKView, CIRenderDestination), macOS 14+ (Sonoma)

## File Structure

```

├── LookbookApp.swift              — @main App entry, WindowGroup, window sizing
├── ContentView.swift              — Top-level HSplitView: preview + right panel
├── MetalView.swift                — NSViewRepresentable wrapping MTKView
├── Renderer.swift                 — MTKViewDelegate, CIContext, CIRenderDestination draw
├── EditingPipeline.swift          — @Observable @MainActor: CIRAWFilter + filter chain + state
├── EditPanel.swift                — SwiftUI sliders for exposure/temp/tint/contrast/saturation
├── LUTPanel.swift                 — LUT browser: thumbnail list, add buttons, intensity slider
├── LUTLoader.swift                — .cube file parser → CIColorCubeWithColorSpace filter
├── LUTThumbnailGenerator.swift    — Background CIContext rendering of per-LUT preview thumbnails
├── ExportManager.swift            — Full-resolution JPEG export via separate CIContext
├── Lookbook.entitlements          — App sandbox + file bookmark entitlements
└── Info.plist                     — (auto-generated by Xcode)
```

**Responsibilities and boundaries:**

- `EditingPipeline` is the single source of truth for all image state (RAW URL, slider values, selected LUT, intensity). It exposes a computed `outputImage: CIImage?` that other components consume. No other file builds filter chains.
- `Renderer` only knows how to draw a `CIImage` to an `MTKView`. It receives images via a closure — it does not import or reference `EditingPipeline`.
- `MetalView` is a thin `NSViewRepresentable` wrapper. It creates the `MTKView` and `Renderer`, wires them together, and handles drag-and-drop.
- `LUTLoader` is a pure parser: `.cube` file URL in, `CIFilter` out. No UI, no state.
- `LUTThumbnailGenerator` takes a base `CIImage` and a list of `CIFilter`s, returns rendered `CGImage` thumbnails. Runs on background thread with its own `CIContext`.
- `ExportManager` is stateless — takes a `CIImage`, quality, and output URL, writes JPEG.
- `EditPanel` and `LUTPanel` are pure SwiftUI views that bind to `EditingPipeline`.

### Task 2: Metal Preview — Renderer

**Files:**
- Create: `Lookbook/Renderer.swift`

This is the core GPU rendering class. It takes a `CIImage` via a closure and draws it to an `MTKView` using `CIRenderDestination` (no CPU-side rasterization).

- [ ] **Step 1: Create Renderer.swift**

```swift
import Metal
import MetalKit
import CoreImage

final class Renderer: NSObject, MTKViewDelegate {
    let device: MTLDevice
    let commandQueue: MTLCommandQueue
    let ciContext: CIContext
    var imageProvider: (() -> CIImage?)?

    init(device: MTLDevice) {
        self.device = device
        self.commandQueue = device.makeCommandQueue()!
        self.ciContext = CIContext(
            mtlDevice: device,
            options: [
                .workingColorSpace: CGColorSpace(name: CGColorSpace.extendedLinearDisplayP3)!,
                .cacheIntermediates: false
            ]
        )
        super.init()
    }

    func mtkView(_ view: MTKView, drawableSizeWillChange size: CGSize) {
        // No-op — we render on demand
    }

    func draw(in view: MTKView) {
        guard let image = imageProvider?(),
              let drawable = view.currentDrawable,
              let commandBuffer = commandQueue.makeCommandBuffer() else {
            return
        }

        let drawableSize = view.drawableSize
        let destination = CIRenderDestination(
            width: Int(drawableSize.width),
            height: Int(drawableSize.height),
            pixelFormat: view.colorPixelFormat,
            commandBuffer: commandBuffer,
            mtlTextureProvider: { () -> MTLTexture in
                return drawable.texture
            }
        )
        destination.colorSpace = CGColorSpace(name: CGColorSpace.extendedLinearDisplayP3)

        // Scale image to fit view, centered
        let scaleX = drawableSize.width / image.extent.width
        let scaleY = drawableSize.height / image.extent.height
        let scale = min(scaleX, scaleY)

        var scaledImage = image.transformed(by: CGAffineTransform(scaleX: scale, y: scale))
        let offsetX = (drawableSize.width - scaledImage.extent.width) / 2 - scaledImage.extent.origin.x
        let offsetY = (drawableSize.height - scaledImage.extent.height) / 2 - scaledImage.extent.origin.y
        scaledImage = scaledImage.transformed(by: CGAffineTransform(translationX: offsetX, y: offsetY))

        do {
            try ciContext.startTask(toRender: scaledImage, to: destination)
        } catch {
            print("CIRenderDestination error: \(error)")
        }

        commandBuffer.present(drawable)
        commandBuffer.commit()
    }
}
```

Key details:
- `CIRenderDestination` renders directly to the MTKView's drawable texture — zero CPU copies.
- `colorPixelFormat` of the view will be `.rgba16Float` (set by MetalView).
- Working color space is `extendedLinearDisplayP3` for wide gamut + EDR.
- Image is scaled to fit the view with aspect ratio preserved, centered.

```swift
        context.coordinator.onDropRAW = onDropRAW
        context.coordinator.metalView = view

        return view

        context.coordinator.onDropRAW = onDropRAW
        nsView.setNeedsDisplay(nsView.bounds)
    }

    func makeCoordinator() -> Coordinator {
        Coordinator()
    }

    class Coordinator: NSObject {
        var onDropRAW: ((URL) -> Void)?
        weak var metalView: MTKView?
    }
}
```

- `framebufferOnly = false` is required for `CIRenderDestination` to write to the drawable texture.
- `colorPixelFormat = .rgba16Float` for wide gamut / EDR support.

### Task 4: EditingPipeline — RAW Loading and Filter Chain

**Files:**
- Create: `Lookbook/EditingPipeline.swift`

The central model object. Owns all image state, builds the lazy CIImage filter chain.

- [ ] **Step 1: Create EditingPipeline.swift**

```swift
import CoreImage
import CoreImage.CIFilterBuiltins
import Observation

@Observable
@MainActor
final class EditingPipeline {
    // MARK: - RAW State
    private(set) var rawFilter: CIRAWFilter?
    private(set) var rawURL: URL?
    private(set) var imageDimensions: CGSize = .zero

    // MARK: - Edit Parameters
    var exposure: Float = 0.0 {
        didSet { applyRAWParameters() }
    }
    var temperature: Float = 6500.0 {
        didSet { applyRAWParameters() }
    }
    var tint: Float = 0.0 {
        didSet { applyRAWParameters() }
    }
    var contrast: Float = 1.0
    var saturation: Float = 1.0

    // MARK: - LUT State
    var selectedLUT: CIFilter?
    var lutIntensity: Float = 1.0

    // MARK: - Export
    var jpegQuality: Float = 0.9

    // MARK: - Computed Output
    var hasImage: Bool { rawFilter != nil }

    /// The full lazy filter chain. No pixels are computed until this CIImage is rendered.
    var outputImage: CIImage? {
        guard let rawFilter = rawFilter,
              var image = rawFilter.outputImage else {
            return nil
        }

        // Apply CIColorControls (contrast + saturation)
        if contrast != 1.0 || saturation != 1.0 {
            let colorControls = CIFilter.colorControls()
            colorControls.inputImage = image
            colorControls.contrast = contrast
            colorControls.saturation = saturation
            if let result = colorControls.outputImage {
                image = result
            }
        }

        // Apply selected LUT
        if let lutFilter = selectedLUT {
            let preBlendImage = image
            lutFilter.setValue(image, forKey: kCIInputImageKey)
            if let lutOutput = lutFilter.outputImage {
                if lutIntensity < 1.0 {
                    // Blend between original and LUT-applied using CIDissolveTransition
                    let dissolve = CIFilter.dissolveTransition()
                    dissolve.inputImage = preBlendImage
                    dissolve.targetImage = lutOutput
                    dissolve.time = lutIntensity
                    if let blended = dissolve.outputImage {
                        image = blended
                    }
                } else {
                    image = lutOutput
                }
            }
        }

        return image
    }

    /// The base RAW decode (no edits applied beyond white balance) for LUT thumbnail generation.
    var baseImage: CIImage? {
        rawFilter?.outputImage
    }

    // MARK: - Load RAW

    func loadRAW(from url: URL) {
        guard let filter = CIRAWFilter(imageURL: url) else {
            print("Failed to create CIRAWFilter for \(url.lastPathComponent)")
            return
        }

        rawURL = url
        rawFilter = filter

        // Enable draft mode for fast preview decoding
        filter.isDraftModeEnabled = true

        // Read metadata defaults for temperature and tint
        temperature = filter.neutralTemperature
        tint = filter.neutralTint
        exposure = 0.0

        // Get image dimensions from the filter's output
        if let output = filter.outputImage {
            imageDimensions = output.extent.size
        }

        applyRAWParameters()
    }

    // MARK: - Private

    private func applyRAWParameters() {
        guard let rawFilter = rawFilter else { return }
        rawFilter.exposure = exposure
        rawFilter.neutralTemperature = temperature
        rawFilter.neutralTint = tint
    }
}
```

Key details:
- `@Observable` (Observation framework, macOS 14+) — SwiftUI views automatically track property access.
- `@MainActor` — all mutations happen on main thread; safe for SwiftUI binding.
- `outputImage` is a computed property that builds the lazy CIImage filter chain on every access. This is cheap — Core Image doesn't compute pixels until render time.
- `isDraftModeEnabled = true` for fast preview decoding (~2-5x faster).
- `CIDissolveTransition` blends between pre-LUT and post-LUT images for the intensity slider. `time` parameter: 0.0 = inputImage (original), 1.0 = targetImage (LUT applied).
- `baseImage` provides the raw decode without edit parameters for thumbnail generation (thumbnails don't update on slider drags — too expensive).

```swift
import SwiftUI
import MetalKit
import UniformTypeIdentifiers

struct MetalView: NSViewRepresentable {
    let renderer: Renderer
    var onDropRAW: ((URL) -> Void)?

    func makeNSView(context: Context) -> DragDropMTKView {
        let view = DragDropMTKView()
        view.device = renderer.device
        view.delegate = renderer
        view.colorPixelFormat = .rgba16Float
        view.colorspace = CGColorSpace(name: CGColorSpace.extendedLinearDisplayP3)
        view.clearColor = MTLClearColor(red: 0.1, green: 0.1, blue: 0.1, alpha: 1.0)
        view.isPaused = true
        view.enableSetNeedsDisplay = true
        view.framebufferOnly = false
        view.onDropRAW = onDropRAW
        return view
    }

    func updateNSView(_ nsView: DragDropMTKView, context: Context) {
        nsView.onDropRAW = onDropRAW
        nsView.setNeedsDisplay(nsView.bounds)
    }
}

class DragDropMTKView: MTKView {
    var onDropRAW: ((URL) -> Void)?

    override init(frame frameRect: CGRect, device: MTLDevice?) {
        super.init(frame: frameRect, device: device)
        registerForDraggedTypes([.fileURL])
    }

    required init(coder: NSCoder) {
        fatalError("init(coder:) not implemented")
    }

    override func draggingEntered(_ sender: NSDraggingInfo) -> NSDragOperation {
        if hasRAWFile(in: sender) {
            return .copy
        }
        return []
    }

    override func performDragOperation(_ sender: NSDraggingInfo) -> Bool {
        guard let url = extractFileURL(from: sender) else { return false }

        // Check if the file is a RAW image by its UTType
        if let type = try? url.resourceValues(forKeys: [.contentTypeKey]).contentType,
           type.conforms(to: .rawImage) {
            onDropRAW?(url)
            return true
        }
        return false
    }

    private func hasRAWFile(in info: NSDraggingInfo) -> Bool {
        guard let url = extractFileURL(from: info) else { return false }
        if let type = try? url.resourceValues(forKeys: [.contentTypeKey]).contentType {
            return type.conforms(to: .rawImage)
        }
        return false
    }

    private func extractFileURL(from info: NSDraggingInfo) -> URL? {
        guard let items = info.draggingPasteboard.readObjects(forClasses: [NSURL.self]) as? [URL],
              let url = items.first else {
            return nil
        }
        return url
    }
}
```

### Task 8: LUT Loader — .cube File Parser

**Files:**
- Create: `Lookbook/LUTLoader.swift`

Parse .cube LUT files into `CIColorCubeWithColorSpace` filters.

- [ ] **Step 1: Create LUTLoader.swift**

```swift
import CoreImage

enum LUTLoaderError: Error, LocalizedError {
    case missingDimension
    case dimensionMismatch(expected: Int, got: Int)
    case invalidData(line: Int, content: String)
    case fileReadError(URL)

    var errorDescription: String? {
        switch self {
        case .missingDimension:
            return "LUT file missing LUT_3D_SIZE declaration"
        case .dimensionMismatch(let expected, let got):
            return "Expected \(expected * expected * expected) RGB entries, got \(got)"
        case .invalidData(let line, let content):
            return "Invalid data at line \(line): \(content)"
        case .fileReadError(let url):
            return "Failed to read file: \(url.lastPathComponent)"
        }
    }
}

struct LUTLoader {
    /// Parse a .cube file and return a CIColorCubeWithColorSpace filter.
    ///
    /// The .cube format spec:
    /// - Lines starting with # are comments
    /// - `TITLE "name"` is optional metadata
    /// - `LUT_3D_SIZE N` declares the cube dimension (required)
    /// - `DOMAIN_MIN r g b` and `DOMAIN_MAX r g b` define the input range (default 0-1)
    /// - Data lines contain three floats: `R G B` (one per lattice point)
    /// - Data is ordered: R changes fastest, then G, then B
    ///
    /// CIColorCubeWithColorSpace expects RGBA float data (4 floats per entry), so we
    /// append alpha = 1.0 to each RGB triple.
    static func parse(cubeFileAt url: URL) throws -> CIFilter {
        guard let content = try? String(contentsOf: url, encoding: .utf8) else {
            throw LUTLoaderError.fileReadError(url)
        }

        var dimension: Int?
        var title: String?
        var rgbData: [Float] = []
        var lineNumber = 0

        for line in content.components(separatedBy: .newlines) {
            lineNumber += 1
            let trimmed = line.trimmingCharacters(in: .whitespaces)

            // Skip empty lines and comments
            if trimmed.isEmpty || trimmed.hasPrefix("#") {
                continue
            }

            // Parse metadata
            if trimmed.hasPrefix("TITLE") {
                title = trimmed
                    .replacingOccurrences(of: "TITLE", with: "")
                    .trimmingCharacters(in: .whitespaces)
                    .trimmingCharacters(in: CharacterSet(charactersIn: "\""))
                continue
            }

            if trimmed.hasPrefix("LUT_3D_SIZE") {
                let parts = trimmed.split(separator: " ")
                if parts.count >= 2, let dim = Int(parts[1]) {
                    dimension = dim
                }
                continue
            }

            // Skip DOMAIN_MIN, DOMAIN_MAX (we assume 0-1 range)
            if trimmed.hasPrefix("DOMAIN_MIN") || trimmed.hasPrefix("DOMAIN_MAX") {
                continue
            }

            // Parse RGB data line
            let parts = trimmed.split(separator: " ")
            if parts.count >= 3,
               let r = Float(parts[0]),
               let g = Float(parts[1]),
               let b = Float(parts[2]) {
                rgbData.append(contentsOf: [r, g, b])
            }
            // Skip lines that don't match any known format (tolerance for non-standard files)
        }

        guard let dim = dimension, dim > 0 else {
            throw LUTLoaderError.missingDimension
        }

        let expectedCount = dim * dim * dim
        let actualCount = rgbData.count / 3
        guard actualCount == expectedCount else {
            throw LUTLoaderError.dimensionMismatch(expected: expectedCount, got: actualCount)
        }

        // Convert RGB to RGBA (CIColorCubeWithColorSpace needs 4 channels)
        var rgbaData: [Float] = []
        rgbaData.reserveCapacity(expectedCount * 4)
        for i in 0..<expectedCount {
            rgbaData.append(rgbData[i * 3])     // R
            rgbaData.append(rgbData[i * 3 + 1]) // G
            rgbaData.append(rgbData[i * 3 + 2]) // B
            rgbaData.append(1.0)                  // A
        }

        // Create CIColorCubeWithColorSpace filter
        let data = Data(bytes: rgbaData, count: rgbaData.count * MemoryLayout<Float>.size)
        let colorSpace = CGColorSpace(name: CGColorSpace.sRGB)!

        let filter = CIFilter(name: "CIColorCubeWithColorSpace")!
        filter.setValue(dim, forKey: "inputCubeDimension")
        filter.setValue(data, forKey: "inputCubeData")
        filter.setValue(colorSpace, forKey: "inputColorSpace")
        filter.setValue(true, forKey: "inputExtrapolate")

        _ = title  // Available for display purposes if needed

        return filter
    }
}
```

Key details:
- `inputColorSpace: sRGB` — CIColorCubeWithColorSpace converts input to sRGB before applying the LUT, then converts back. This is correct for standard .cube files which assume sRGB input.
- `inputExtrapolate: true` — preserves EDR highlights that extend beyond 0-1 range.
- RGBA format: CIColorCubeWithColorSpace requires 4 floats per entry (not 3). We append alpha = 1.0.
- `inputCubeDimension` is the cube size N (e.g., 33 or 64). Total entries = N^3.

```swift
import CoreImage
import CoreGraphics

struct LUTThumbnailGenerator {
    /// Shared CIContext for background thumbnail rendering.
    /// Separate from the main Renderer's CIContext to avoid contention.
    private static let ciContext = CIContext(options: [
        .workingColorSpace: CGColorSpace(name: CGColorSpace.sRGB)!,
        .cacheIntermediates: false
    ])

    /// Maximum thumbnail width/height in pixels.
    private static let maxSize: CGFloat = 120

    /// Generate a CGImage thumbnail by applying a LUT filter to a base image.
    ///
    /// - Parameters:
    ///   - baseImage: The raw decoded CIImage (should be at reduced scale for performance)
    ///   - lutFilter: The CIColorCubeWithColorSpace filter to apply
    /// - Returns: A small CGImage suitable for display in a LUT card
    static func generateThumbnail(from baseImage: CIImage, with lutFilter: CIFilter) -> CGImage? {
        // Scale down to thumbnail size
        let scale = min(maxSize / baseImage.extent.width, maxSize / baseImage.extent.height, 1.0)
        var image = baseImage.transformed(by: CGAffineTransform(scaleX: scale, y: scale))

        // Apply LUT
        lutFilter.setValue(image, forKey: kCIInputImageKey)
        if let output = lutFilter.outputImage {
            image = output
        } else {
            return nil
        }

        // Render to CGImage
        return ciContext.createCGImage(image, from: image.extent)
    }
}
```
